¿Qué es GitHub?

Tu portafolio vivo en la nube y una herramienta clave para el trabajo en equipo

Definición

GitHub es una plataforma en línea para alojar repositorios Git. Permite gestionar proyectos, colaborar con otros desarrolladores, compartir código y trabajar en equipo desde cualquier parte del mundo.

¿Para qué se usa?

Se utiliza para almacenar y compartir código fuente, realizar seguimiento de cambios, crear documentación, gestionar tareas y colaborar en proyectos de desarrollo de software. También es el lugar donde muchos proyectos open-source viven y evolucionan.

Características principales

  • Repositorios públicos y privados.
  • Control de versiones basado en Git.
  • Pull requests para revisión de código.
  • Issues y proyectos para gestión de tareas.
  • Integración continua (CI/CD) con GitHub Actions.

¿Por qué es importante en tu portafolio?

Porque tus repositorios muestran tu forma de programar, cómo estructuras tus proyectos, qué tecnologías dominas y cómo documentas tu trabajo. Un buen perfil de GitHub puede ayudarte a conseguir oportunidades profesionales.
