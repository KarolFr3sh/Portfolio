¿Qué es HTML5?

La base de toda página web moderna

Definición

HTML5 (HyperText Markup Language, versión 5) es el lenguaje estándar utilizado para crear y estructurar el contenido en la web. Es la versión más reciente de HTML y permite definir elementos como encabezados, párrafos, imágenes, videos, formularios y mucho más.

¿Para qué sirve?

HTML5 se utiliza para estructurar correctamente el contenido de una página web. Esto incluye desde lo más básico como textos, títulos e imágenes, hasta elementos multimedia, gráficos, formularios interactivos y APIs que permiten experiencias modernas en el navegador.

Características principales

  • Soporte nativo para video y audio sin plugins externos.
  • Mejores formularios con nuevos tipos de entrada (email, fecha, rango, etc.).
  • APIs modernas como geolocalización, almacenamiento local, canvas y drag & drop.
  • Semántica mejorada con nuevas etiquetas como <header>, <section>, <article> y <footer>.
  • Compatibilidad con móviles y pantallas pequeñas.

Ejemplo básico de HTML5

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Hola mundo</h1>
    <p>Esta es mi primera página con HTML5.</p>
  </body>
</html>
