¿Qué es Node.js?

El motor que impulsa JavaScript en el servidor

Definición

Node.js es un entorno de ejecución de JavaScript del lado del servidor. Permite utilizar JavaScript fuera del navegador, principalmente para construir aplicaciones backend como servidores web, APIs y sistemas en tiempo real.

¿Para qué sirve?

Con Node.js puedes construir todo tipo de aplicaciones backend, desde APIs REST hasta plataformas en tiempo real como chats o juegos. Gracias a su arquitectura no bloqueante y su motor V8 de Google, ofrece gran rendimiento y escalabilidad.

Ventajas principales

  • Usa JavaScript en el servidor, unificando lenguaje entre frontend y backend.
  • Es rápido gracias a su arquitectura basada en eventos y al motor V8.
  • Cuenta con NPM, el gestor de paquetes más grande del mundo.
  • Ideal para aplicaciones en tiempo real como chats y notificaciones.
  • Gran comunidad y documentación accesible.

Ejemplo básico

// Crear un servidor básico con Node.js
const http = require('http');

const servidor = http.createServer((req, res) => {
  res.end('¡Hola desde Node.js!');
});

servidor.listen(3000, () => {
  console.log('Servidor en http://localhost:3000');
});
