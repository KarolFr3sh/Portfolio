¿Qué es CSS?

El lenguaje que le da estilo a la web

Definición

CSS (Cascading Style Sheets - Hojas de Estilo en Cascada) es el lenguaje que se utiliza para describir la presentación de una página web. A través de CSS, controlamos colores, fuentes, tamaños, márgenes, disposición de elementos y animaciones, haciendo que el contenido HTML cobre vida visualmente.

¿Para qué sirve?

Sin CSS, las páginas web serían solo texto plano. Con CSS, definimos la estética y experiencia visual del usuario: diseño responsive, temas oscuros, animaciones interactivas y estilos adaptados a dispositivos móviles.

Características clave

  • Separa el contenido (HTML) del diseño (CSS).
  • Permite reutilizar estilos en múltiples páginas.
  • Admite media queries para adaptar el diseño a distintas pantallas.
  • Facilita animaciones, transiciones y efectos visuales modernos.
  • Es compatible con múltiples frameworks como Tailwind, Bootstrap o Sass.

Ejemplo básico de CSS

h1 {
  color: #f65f59;
  font-size: 2rem;
  text-align: center;
}

p {
  color: #ccc;
  font-size: 1rem;
}
