¿Qué es JavaScript?

El lenguaje que le da vida a la web

Definición

JavaScript es un lenguaje de programación interpretado que permite crear páginas web interactivas. Junto con HTML y CSS, forma la base del desarrollo web moderno. Gracias a JavaScript podemos responder a eventos, actualizar contenido dinámicamente y crear experiencias de usuario ricas y fluidas.

¿Para qué sirve?

JavaScript permite que los sitios web sean interactivos. Se usa para validar formularios, mostrar mensajes en tiempo real, crear sliders, juegos, animaciones, efectos, y mucho más. Además, también se utiliza en el desarrollo backend (por ejemplo, con Node.js).

Características clave

  • Lenguaje orientado a objetos, eventos y funciones.
  • Funciona en todos los navegadores modernos.
  • Permite modificar el DOM en tiempo real.
  • Compatible con librerías y frameworks como React, Vue, Angular, etc.
  • Se puede usar también del lado del servidor (Node.js).

Ejemplo básico

// Mostrar un mensaje en consola
console.log("¡Hola desde JavaScript!");

// Cambiar el contenido de una etiqueta h1
document.querySelector("h1").textContent = "Texto actualizado con JS";
