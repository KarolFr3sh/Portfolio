¿Qué es Git?

Tu mejor aliado para controlar versiones y trabajar en equipo

Definición

Git es un sistema de control de versiones distribuido creado por Linus Torvalds (creador de Linux). Se utiliza para gestionar y registrar los cambios en el código fuente a lo largo del tiempo, permitiendo trabajar en equipo de manera ordenada.

¿Para qué sirve?

Git permite llevar un historial completo del código, volver a versiones anteriores, trabajar en diferentes ramas (features) y colaborar con otros desarrolladores sin sobreescribir el trabajo de nadie.

Ventajas clave

  • Trabaja de forma local y remota (ideal para proyectos personales y colaborativos).
  • Permite ramificar y fusionar código fácilmente.
  • Es rápido, confiable y ampliamente utilizado en la industria.
  • Se integra perfectamente con plataformas como GitHub, GitLab o Bitbucket.

Comandos básicos

# Inicializar un repositorio
git init

# Ver el estado del repositorio
git status

# Añadir archivos al área de staging
git add .

# Confirmar los cambios (commit)
git commit -m "Mensaje"

# Conectar con un repositorio remoto
git remote add origin URL

# Subir cambios al repositorio
git push -u origin main
